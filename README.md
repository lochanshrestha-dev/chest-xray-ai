# Chest X-ray AI Project

## Project Setup

1. **Clone the Repository**  
   Use the following command to clone the repository:  
   ```bash
   git clone https://github.com/lochanshrestha-dev/chest-xray-ai.git
   cd chest-xray-ai
   ```  

2. **Install Dependencies**  
   Make sure you have Python installed. Then, install the required dependencies:  
   ```bash
   pip install -r requirements.txt
   ```  

## Usage Instructions  

1. **Run the Model**  
   You can run the AI model using the following command:  
   ```bash
   python main.py
   ```  

2. **Testing the Model**  
   To test the model with a sample X-ray image, use:  
   ```bash
   python test.py --image <path_to_your_image>
   ```
